# Multiverse Codex Project Vision

## Phase 1 vision seed

Multiverse Codex is a creator-owned fantasy multiverse website and admin system.

The project direction is:

- self-hosted
- local-first during development
- native Linux deployment without Docker as the primary path
- public lore browsing for readers
- private creator tools for managing canon
- durable data, media, and relationships
- phase-built with proof at every seam

## Product shape

The long-range product is a polished codex for:

- worlds
- characters
- artifacts
- factions
- stories
- chapters
- timeline events
- media assets
- entity relationships
- search
- admin publishing flows
- 3D cosmic exploration

## Technical direction

The canonical plan currently targets:

```txt
SvelteKit
TypeScript
TailwindCSS
Three.js
PostgreSQL
Drizzle ORM
Custom session auth
Caddy
systemd
Ubuntu Server style native deployment
```
